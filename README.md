# 注入攻擊 (Injection)

資料隱碼攻擊：夾帶(惡意指令)讓資料庫伺服器以為是正常SQL指令

# 無效身分認證 (Broken Authentication)

網站應用程式中自行撰寫的身分驗證相關功能有缺陷 (如登入時未加密、密碼強度過弱)

# 敏感資料外洩 (Sensitive Data Exposure)

應用程序未充分保護敏感信息時，會發生敏感數據暴露。數據可以變化，從密碼，信用卡數據到私人健康數據等等都可以暴露出來。
